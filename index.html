国内也能用 Gemini 了！超详细教程

发表于 2025-01-06 分类于 Gemini

国内也能用上 Gemini 了！超详细教程，手把手教你

大家好！是不是早就听说 Google 的 Gemini 很厉害，但又苦于国内用不了？别担心，今天我就来教大家一个超简单的方法，让你在国内也能免费用上 Gemini！不用魔法，也不需要复杂的配置，跟着我的步骤，你也能轻松体验 Gemini 的强大功能。

简单来说，我们要做什么？

其实，我们要做的就是搭建一个“桥梁”，让国内的网络可以连接到 Gemini 的 API，就像搭个梯子一样。这个“梯子”需要借助一个叫 Cloudflare 的工具，还有一些代码，但别怕，我都会一步一步教你。

准备工作，先来个“装备清单”：

  1. Gemini API 密钥： 这是你使用 Gemini 的“通行证”。你需要去 Google AI Studio 申请一个。
    • 打开 aistuidio.google.com，登录你的 Google 账号。
    • 在 Google AI Studio 中，创建一个新的 API 密钥。记住，即使你不在支持的地区，也可以用 VPN 申请！
  2. Cloudflare 账号： 你需要注册一个 Cloudflare 账号，它是我们搭建“梯子”的重要工具。
    • 打开 cloudflare.com ，注册一个账号，过程很简单。
  3. Github 账号： 你需要一个 Github 账号，用来“复制”一些代码。
    • 如果你还没有，去 github.com 注册一个吧。
  4. Chatbox 软件： 这是一个可以让你和 Gemini “聊天”的工具，就像微信一样。
    • 去 github.com/Bin-Huang/chatbox 下载并安装它。

第一步：搭建“梯子”——部署 Cloudflare Worker

  1. “复制”代码：
    • 打开 github.com/PublicAffairs/openai-gemini 这个链接，这是“梯子”的代码来源。
    • 点击右上角的 “Fork” 按钮，把这个代码“复制”到你的 Github 账号里。
  2. “提取”代码：
    • 在你 Github 的“复制”项目里，找到 src/worker.mjs 这个文件。
    • 复制这个文件里的所有代码。
  3. “搭建”梯子：
    • 登录你的 Cloudflare 账号，找到 “Workers & Pages” 页面。
    • 点击 “创建 Worker” 按钮，就像新建一个文件一样。
    • 把你刚才复制的代码粘贴到代码编辑器里。
  4. “配置”通行证：
    • 在代码编辑器下方，找到 “设置” 选项卡，点击 “环境变量”。
    • 添加一个叫 GOOGLE_API_KEY 的环境变量，把你的 Gemini API 密钥填进去。
  5. “架好”梯子：
    • 点击 “保存并部署” 按钮，你的 Cloudflare Worker 就会被部署，就像搭好了一个梯子一样。
    • 复制你的 Worker 的网址，比如 https://your-worker-name.your-cloudflare-subdomain.workers.dev，这个网址很重要，后面要用到！

第二步：让 Chatbox 和 Gemini “连上线”

  1. 打开 Chatbox： 启动你安装的 Chatbox 软件。
  2. 添加 Gemini 模型：
    • 在 Chatbox 的设置里，找到“模型”或者“自定义模型”的选项。
    • 点击“添加模型”或类似的按钮。
  3. “告诉”Chatbox 怎么连：
    • 模型名称： 可以随便起，比如 “Gemini-Cloudflare”。
    • API 类型： 选择 OpenAI。
    • API Base URL： 把你刚才复制的 Cloudflare Worker 网址粘贴到这里，并在末尾加上 /v1，比如 https://your-worker-name.your-cloudflare-subdomain.workers.dev/v1。
    • API Key： 这里不需要填，因为 Gemini 的密钥已经配置在 Cloudflare Worker 里了。
    • 模型名称： 填入 gemini-1.5-pro-latest 或者 models/gemini-1.5-pro-latest。
  4. 测试 “连接”：
    • 点击 “测试连接” 或类似的按钮，确保 Chatbox 可以成功连接到你的 Gemini 代理。

第三步：开始和 Gemini “聊天”

  1. 选择 Gemini： 在 Chatbox 里，选择你刚刚添加的 Gemini-Cloudflare 模型。
  2. 开始对话： 现在，你可以像使用 OpenAI 一样，和 Gemini 模型进行对话了！

温馨提示：

  • 模型选择： 你可以使用 gemini-1.5-pro-latest 模型，或者其他 Gemini 模型。
  • API 限制： Gemini API 可能有使用次数限制，注意查看 Google AI Studio 的文档。
  • Cloudflare Worker 限制： Cloudflare Worker 也有一些限制，比如运行时间限制，如果你的请求太多可能会遇到问题。
  • openai-gemini 项目： 这个项目还在开发中，可能会有一些小问题，请关注它的更新。

总结一下：

通过以上步骤，你已经成功搭建了一个“桥梁”，让国内的网络可以连接到 Gemini 的 API，并用 Chatbox 和 Gemini “聊上了天”！是不是很简单？快去试试吧，体验一下 Gemini 的强大吧！
